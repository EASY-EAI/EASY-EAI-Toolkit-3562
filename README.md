如果您初次接触本项目，您可以：  
[查阅入门(快速上手)指南](https://www.easy-eai.com/document_details/33/1240)

如果您具备丰富的嵌入式C语言开发经验，您可以：  
[查阅EASYEAI开发实例](https://www.easy-eai.com/document_details/33/1314)  
[查阅API开发手册](https://www.easy-eai.com/document_details/33/1295)


使用方法：  
[1] - 克隆本GitHub仓库   
[2] - 进入具体的(demo/solution)目录执行：./build.sh   
[3] - 则会在具体的(demo/solution)目录的Release/下生成相关的可执行程序
